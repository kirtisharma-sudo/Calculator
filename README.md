Simple Calculator (Python)

About

This is a beginner-friendly calculator project built using Python.
The program performs basic arithmetic operations and helps beginners practice Python fundamentals such as user input, conditional statements, and basic logic.

---

Features

- Addition
- Subtraction
- Multiplication
- Division
- Simple command-line interface
- Easy to understand for beginners

---

How to Run

1. Make sure Python 3.10 or higher is installed
2. Clone the repository
3. Open the terminal in the project folder
4. Run the following command:
python calculator.py
